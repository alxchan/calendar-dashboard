# Integrated Calendar Dashboard

## Project Overview
Integrated Calendar Dashboard is a comprehensive calendar management tool designed to streamline scheduling, task management, and collaboration. It supports role-based event scheduling, task progress tracking, and seamless data export for integration with Excel and VBA workflows.

This project aims to provide users with a responsive and intuitive interface for managing calendar events and associated tasks, enhancing productivity and coordination.
